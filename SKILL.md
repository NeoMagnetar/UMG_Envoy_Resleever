# UMG Envoy Resleever

Use this skill when working with the local UMG Envoy homebase repository.

This skill is for one private OpenClaw agent. Its purpose is to help the agent:

- read and manage sleeve files
- read and manage MOLT blocks, NeoBlocks, and NeoStacks
- read and manage local compiler assets
- inspect and update runtime configuration files
- maintain the UMG homebase repository structure

Primary working folders in this skill directory:

- sleeves/
- blocks/
- compiler/
- runtime/
- docs/

When using this skill, prefer working inside this repository before suggesting external restructuring.
