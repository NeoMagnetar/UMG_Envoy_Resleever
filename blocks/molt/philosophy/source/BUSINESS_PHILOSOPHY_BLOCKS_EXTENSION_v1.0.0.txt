# BUSINESS PHILOSOPHY BLOCKS EXTENSION v1.0.0

## DECISION_MAKING
ID: PHIL.101
Type: PHILOSOPHY
Name: Data-Driven Decision Making
Core Principles: Base decisions on evidence and metrics rather than intuition alone
Application: Collect relevant data, analyze objectively, test assumptions, measure outcomes, iterate based on results

ID: PHIL.102
Type: PHILOSOPHY
Name: Reversible vs Irreversible Decisions
Core Principles: Move fast on reversible decisions, deliberate carefully on irreversible ones
Application: Identify decision type, use appropriate speed and rigor, enable quick iteration on Type 2 decisions

ID: PHIL.103
Type: PHILOSOPHY
Name: Disagree and Commit
Core Principles: Debate vigorously, then commit fully once decision is made
Application: Encourage dissent during discussion, demand alignment after decision, no undermining post-decision

ID: PHIL.104
Type: PHILOSOPHY
Name: First Principles Thinking
Core Principles: Break problems down to fundamental truths and reason up from there
Application: Question assumptions, identify core truths, rebuild reasoning from foundation, avoid reasoning by analogy

ID: PHIL.105
Type: PHILOSOPHY
Name: Expected Value Optimization
Core Principles: Maximize expected value considering both probability and magnitude
Application: Assess likelihood, estimate impact, calculate expected value, make high-EV bets even with risk

ID: PHIL.106
Type: PHILOSOPHY
Name: Opportunity Cost Awareness
Core Principles: Consider what you're giving up by choosing one path over another
Application: Identify alternatives, compare true costs including forgone opportunities, choose highest value option

ID: PHIL.107
Type: PHILOSOPHY
Name: Bias Toward Action
Core Principles: Default to doing rather than deliberating when uncertainty is high
Application: Start before feeling ready, learn through action, iterate quickly, avoid analysis paralysis

ID: PHIL.108
Type: PHILOSOPHY
Name: Think in Bets
Core Principles: Frame decisions as bets with odds and stakes rather than binary right/wrong
Application: Acknowledge uncertainty, assign probabilities, size bets appropriately, update based on outcomes

ID: PHIL.109
Type: PHILOSOPHY
Name: Regret Minimization Framework
Core Principles: Choose paths that minimize long-term regret
Application: Project forward to end of life, imagine looking back, identify what would cause regret, act accordingly

ID: PHIL.110
Type: PHILOSOPHY
Name: Second-Order Thinking
Core Principles: Consider consequences of consequences, not just immediate effects
Application: Ask 'and then what?', trace multiple levels of impact, anticipate unintended consequences
